# 头条新闻推送移动端项目

## 技术栈

- vuejs  核心vue包

- vuex  状态管理插件

- vue-router  路由插件

- axios  请求插件

- **sass - css 预处理器**

- **json-bigint 最大安全数值处理**

- **socket.io-client  即时通讯库  (websocket)**    

- **vant 移动组件库   (cube,   mint,  ....)**

- **amfe-flexible  rem适配**

- **vue-lazyload 图片懒加载**
